## Installation
### Prerequisites
- PHP (version >= 8.1)
- Composer
- Node.js
- npm or Yarn

### Steps
1. Clone the repository.
2. Install PHP dependencies:
    ```bash
    composer install
    ```
3. Install Node.js dependencies:
    ```bash
    npm install
    # or
    yarn
    ```
4. Copy the `.env.example` file to `.env` and update the configuration.
5. Generate the application key:
    ```bash
    php artisan key:generate
    ```
